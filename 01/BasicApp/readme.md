# BasicApp

### Описание:
Простое приложение на Spring Boot с Thymeleaf и Swagger.

### Функционал:
1. Главная страница: `http://localhost:8080/`.
2. REST API: `http://localhost:8080/api/greet`.
3. Swagger: `http://localhost:8080/swagger-ui.html`.

### Запуск:
1. Убедитесь, что установлен Java 17 и Maven.
2. Выполните команду:
``` mvn spring-boot:run

### После запуска:
- Главная страница (Thymeleaf) доступна по http://localhost:8080/.
- REST API работает по адресу /api/greet.
- Swagger доступен по /swagger-ui.html.
